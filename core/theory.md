# Theory
# 本原展开论 core theory v2.0

本原展开论当前核心候选：

```text
本原统一态 / 本原存在（旧称：质点存在）
→ 法则属性 / 展开属性
→ 法则展开
→ 时空
→ 时空中的存在
→ 具体结构
→ 本原张力（本原—展开桥接层）
→ 生命 / 主体桥接条件
→ 主体
→ 认同
→ 希望
→ 协同展开 / 更高认同候选
```

核心中轴句：

```text
存在借展开显现，展开依局限成形，认同凭连续存续，发展使认同更新。
```

## 基本判断

1. 本原统一态 / 本原存在不是时空中的对象，而是前时空、前分化、前对象化的本原候选。
2. 法则属性与展开属性在本原层不应被理解为两个已分离部件，而是在法则展开中显现为可区分的结构约束与生成面向。
3. 法则展开是本原统一态以其统一潜能进入显现层的过程；时空是这一展开的结果与显示框架，而非预设容器。
4. 本原张力是结构约束与生成面向之间，使存在得以成形、持续显现并保持开放可能性的基本关系结构。
5. 时空中的存在是真实展开结构，不是虚假幻象。
6. 结构、生命、主体、认同、希望与协同展开是不同层级，不可跨层直连。

## 关键区分

- 展开属性 ≠ 法则展开；
- 展开属性 ≠ 能量 / 意志 / 希望 / 必然进步；
- 认同 ≠ 希望；
- 主体 ≠ 人类专属，也 ≠ 一切存在；
- 希望 ≠ 乐观、成功期待或情绪安慰；
- 显现层科学接口 ≠ 本原展开论证明；
- Hermes / AI 协同节点 ≠ 完整主体、生命或内在体验。

## 理论应用原则

任何从本原层或高层概念应用到具体现实时，必须经过：

1. **层级桥接**：说明中间层级与转换条件；
2. **接口映射**：说明现实对象如何作为有限、可撤回、受科学约束的候选接口；
3. **失败约束**：说明何种情况会构成解释不足、层级错误、映射失败或概念需要收缩。

## 理论自身原则

本原展开论本身也具有连续性和局限性：

- 连续性：修正应继承已有稳定结构，而不是任意断裂；
- 局限性：必须承认当前无法解释的问题；
- 开放性：允许更高解释结构进入；
- 反思性：持续检查概念漂移、层级混淆、绝对化和封闭化；
- 现实拟合：显现层必须接受科学事实、数学模型和经验约束。

理论可以保留 unknown，但不能以 unknown 否认现实。理论可以使用科学接口，但不能用哲学概念替代具体科学机制。
