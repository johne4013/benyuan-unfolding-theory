# 心脏外科 日常病程记录智能生成 —— 设计方案

## 一、定位

科室文书工具链第三块（出院告知 → 入院采集 → 病程记录）。第一版聚焦**日常病程记录**
——病程记录中最高频、最重复的一种。沿用入院工具已验证模式：单文件零依赖网页、点选
为主、自动成文、纯文本框输出 + 复制按钮，复制粘贴到内部系统（HIS/电子病历）使用；
无患者身份信息、无打印排版、`localStorage` 暂存。

- 形态：`progress-note/index.html`，与 `cardiac-discharge/`、`admission/` 并列。

## 二、生成逻辑

按《病历书写基本规范》日常病程记录要素，点选 → 按规范句式自动成段：

| 要素 | 录入 |
|------|------|
| 记录日期时间 / 术后第 N 天 | 字段（时间默认当前） |
| 手术/术式 | 下拉，联动追加查体与处理要点 |
| 主观 / 一般情况 | 点选（一般情况、切口疼痛、精神饮食睡眠二便等） |
| 查体 | 生命体征（可留空）+ 点选（切口、心、肺、腹、下肢、引流管、起搏器等） |
| 辅助检查 | 今复查项目点选 + 结果自由填（留空生成“结果回报：____”） |
| 病情分析 | 点选（平稳/好转/恢复顺利/需观察等） |
| 诊疗计划 | 点选（继续治疗、抗感染、强心利尿、排痰、拔管、换药、出院等）+ 上级医师查房 |

输出为一段规范日常病程，带签名行；____ 占位提醒补全。

## 三、术式联动

选择术式后，自动向"查体"与"诊疗计划"追加该术式相关要点，例如：

- 机械瓣置换 → 处理追加"监测 INR、调整华法林剂量"；
- 冠脉搭桥(CABG) → 查体追加"取大隐静脉侧下肢切口愈合好"，处理追加"患肢穿弹力袜、抗血小板"；
- TEVAR / 大血管 → 处理追加"严格控制血压、心率"；
- 心包开窗/引流 → 查体追加引流管、处理追加"记录每日引流量"。

## 四、合规说明

- 生成内容为**辅助草稿**，复制入内部系统后须由经治医师核对、修改并签字；注意补全
  ____ 占位。日常病程记录按规范应及时书写（病危每日、病重视病情、病情稳定者至少每
  2~3 天一次，遵院内要求）。
- 纯本地运行，患者数据不出科室电脑。
- 模板（各组点选条目、术式联动）集中在文件顶部 `SUBJ`/`BASE_EXAM`/`RECHECK`/
  `ANALYSIS`/`BASE_PLAN`/`ROUNDS`/`SURG` 数据区，可自行增删。

## 五、后续可扩展（暂未做）

首次病程记录（病例特点/拟诊讨论/诊疗计划）、术前小结、术后首次病程记录、上级医师
查房记录等其他病程类型，可按相同模式逐步追加为多 Tab。
