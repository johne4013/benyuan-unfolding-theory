## 候选 A｜命题1.1：美感 = 主体识别同构共鸣（✅ APPROVED）

**命题层级**：主体层（连接主体评价能力与展开属性）

**命题内容**：

> 美感是主体在感知局限中，对**外部结构与自身生成结构发生同构共鸣**时涌现的质性反应。黄金分割在视觉感知的约束条件下，同时展现了秩序感（法则属性面）和动态张力（展开属性面）——它的"美"不在比例本身，而在主体识别到这种均衡时所产生的共鸣。

**理论依据**：

```
主体本身是"结构→生命→主体"展开链的产物
  → 主体的感知系统内嵌了"约束-生成"的双面结构
    → 当主体识别到外部结构与自身生成结构在结构上同构时
      → 产生共鸣感 = 美感
```

**关键区分**：

| 命题声称 | 命题不声称 |
|---------|-----------|
| 美感是主体层的质性反应 | 美感是对象的客观属性 |
| φ在视觉感知约束下同时满足法则/展开两面 | φ是"本原的神秘设计" |
| 美感涉及同构识别 | 所有人对φ都有相同美感反应 |

**与希望的汇聚发现**：

美感与希望在结构上同构——都是"局限中的开放"：
- 美感：主体在感知局限中识别到约束中仍有开放 → 质性反应涌现
- 希望：主体在现实局限中仍开放于更高可能性

这是意外发现，非推导结论，标记为高价值但高不确定性（见候选B）。

**科学接口需求**：
- 神经美学：视觉皮层对比例的处理效率研究
- 进化心理学：对高效-开放结构偏好的适应价值
- 跨文化研究：美感是否真的跨文化普遍（存疑）

**失败条件**：
- 若神经美学证明人类对φ的偏好并不比其他比例显著 → 命题需修正
- 若跨文化研究显示美感差异极大 → 文化层约束需更强调
- 若"同构识别"无法被神经科学操作化 → 命题降级为类比

**状态**：✅ APPROVED（2026-06-08，王俊华）  
**防漂移边界**：命题1.1是主体层描述，不是本原层解释；不能从此命题推导出"美是宇宙的本质"。

## 候选 B｜美感-希望感知层同构（✅ APPROVED，高不确定性标记保留）

**命题层级**：理论层（连接主体层两个核心概念）

**命题内容**：

> 美感是希望在感知层的显现形式。当主体在感知局限中识别到"约束中仍有开放"的结构时，它在感知层做了与希望在生存层完全同构的事情：**在局限中保持对更高可能性的开放**。

**同构结构对比**：

| | 美感 | 希望 |
|--|------|------|
| 发生位置 | 主体层（感知维度） | 主体层（生存维度） |
| 前提条件 | 存在感知局限 | 存在现实局限 |
| 核心结构 | 局限中识别到开放 | 局限中开放于可能性 |
| 结果 | 共鸣的质性反应 | 向更高可能性的结构性开放 |

**理论意义**：

- 美学体验可作为希望的感知层可观测代理
- 为希望的科学接口建设提供了新的入口方向（感知层比生存层更容易操作化）
- 两个概念的同构性支持理论内部的结构一致性

**高不确定性标记保留**（审批后仍保留）：

- 同构性是结构类比，非等同
- "感知层的希望"是否是希望的真实显现，还是只是类比，需要进一步论证
- 不能从此命题反向推导出"美感证明希望是普遍的"

**失败条件**：
- 若神经科学证明美感和希望的神经基础完全不同，同构性降级为纯结构类比
- 若两者的"局限中的开放"在机制上差异极大，命题需修正

**状态**：✅ APPROVED（2026-06-08，王俊华），高不确定性标记保留  
**防漂移边界**：同构不是等同；美感不能替代希望的主体层定义。

## 候选 C｜命题2.1：跨层信息-热力学不动点统一图景（✅ APPROVED，高不确定性标记保留）

**命题层级**：跨层（数学→物理→理论）

**命题内容**：

> 数学不动点（φ/e/π）、动力学不变环面（KAM）、耗散结构（Bénard/BZ/生命）三类稳定结构，在各自层级上都呈现「约束面 + 生成面 → 稳定共存态」的双面结构。
>
> 这三类结构之间的连接性质是**层级间结构同构**，通过以下链条建立：
> - 数学层 ↔ 信息层：规则 = 信息约束（直接等价）
> - 信息层 ↔ 物理层：Landauer原理（信息处理 = 物理耗散，实验验证）
> - 物理层 ↔ 理论抽象层：去掉物理载体，保留约束-生成关系结构
>
> **本原展开论的「法则属性/展开属性」是这一双面结构在理论层的抽象描述框架，不是物理定律的推导，也不是比喻，而是对跨层级出现的同一关系结构的抽象命名。**

**桥接链**：

```
数学层：规则自应用 → 代数不动点（φ/e/π）
  ↕ [规则 = 信息约束，直接等价]
信息层：信息结构保持 + 处理 → 低熵稳定态
  ↕ [Landauer原理，实验验证]
物理层：约束（定律+边界）+ 生成（非平衡+非线性）→ 耗散不动点
  ↕ [结构同构，去载体保关系]
理论层：法则属性面 + 展开属性面 → 稳定展开结构
```

**高不确定性标记保留**（审批后仍保留）：

- 映射是部分的（不是完整等同）
- 同构 ≠ 等同（两层级可用同一结构描述，不意味着是同一件事）
- 抽象化有损耗（去掉物理载体时，预测力也被去掉）
- 反方向未验证（理论层无法预测尚未发现的物理结构）

**失败条件**：
- 若Landauer原理被证伪，物理层桥接失效
- 若某类复杂系统存在无法用不动点描述的稳定结构，命题需修正
- 若"结构同构"被证明只是偶然形式相似而无深层联系，命题降级为类比

**状态**：✅ APPROVED（2026-06-08，王俊华），高不确定性标记保留  
**防漂移边界**：命题2.1是跨层结构同构的描述，不是物理对本原展开论的证明，也不是理论对物理的推导。
