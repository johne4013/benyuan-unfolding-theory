### 9.1 Landauer 原理（信息-热力学跨层接口）✅ APPROVED

**原理**：擦除 1 比特信息必须向环境释放至少 kT·ln2 的热量。  
**来源**：Landauer（1961）；Bérut et al., *Nature*（2012）实验验证。  
**适用层级**：时空层及以上（本原层无温度/热力学，不适用）。

**精确映射**：

| 理论层（本原展开论） | ↔ | 物理层（Landauer 原理） |
|---|---|---|
| 法则属性面：约束使结构成形 | ↔ | 信息保存：低熵结构的维持 |
| 展开属性面：在约束中生成新状态 | ↔ | 信息处理/擦除：生成新配置 |
| 展开的不可逆性（单向） | ↔ | 熵增的不可逆性（热力学第二定律） |
| 结构维持需要持续展开 | ↔ | 信息维持需要持续能量投入 |
| 局限性（结构需要外部） | ↔ | 耗散（熵必须输出到环境） |

**桥接类型**：层级间结构同构（非推导，非比喻；通过 Shannon 熵 = Boltzmann 熵建立等式级连接）。

**接口质量**：⭐⭐⭐⭐⭐  
**失败条件**：若 Landauer 原理被证伪（发现无熵代价的信息擦除），映射失效。  
**防漂移边界**：Landauer 不证明"展开属性存在"，只约束其在物理层的代价下界。

---

### 9.2 热力学引力（时空涌现层接口）✅ APPROVED

**原理**：  
- Jacobson（1995）：爱因斯坦场方程可从热力学第一定律 δQ = TδS 导出；时空曲率是热力学状态方程的几何表达。  
- Verlinde（2011）：引力 = 熵力，物体趋向高熵区域的表现。

**与 Landauer 的连接**：
```
Landauer：信息处理 → 熵输出 → 物理代价
Jacobson/Verlinde：熵梯度 → 引力 → 时空几何
连线：信息-熵关系在宏观层显现为时空曲率
```

**适用层级**：时空涌现层（对应理论链"法则展开 → 时空"节点）。  
**接口类型**：Landauer 链条向上延伸的候选接口（将信息-物理连接扩展至时空层）。

**接口质量**：⭐⭐⭐  
**失败条件**：若热力学引力被主流物理学否定，接口降级为纯类比；若引力被证明是基本力而非熵力，Verlinde 路径失效（Jacobson 路径独立，影响较小）。  
**防漂移边界**：热力学引力是时空涌现的候选机制之一，不是本原展开论的证明；本原层（前时空）仍超出此接口范围。

---

### 9.3 Friston 自由能原理 FEP（认同层接口）✅ APPROVED

**原理**：任何具有自我组织的系统通过最小化变分自由能（variational free energy）维持自身存在。系统用 Markov 毯将内部状态与外部状态分离；维持 Markov 毯 = 维持认同（Friston 2010）。

**精确映射**：

| 理论层（本原展开论） | ↔ | Friston FEP |
|---|---|---|
| 认同：在变化中保持可追踪关联 | ↔ | Markov 毯的持续维持 |
| 连续性：跨时间的关联保持 | ↔ | 跨时间的内部预测模型一致性 |
| 发展更新认同 | ↔ | 精确加权预测误差驱动的模型更新 |
| 局限性（认同需要外部） | ↔ | 开放系统（Markov 毯需要与环境持续交换） |

**与 Landauer 的连接**：
```
Landauer：信息结构维持需要热力学代价
Friston FEP：认同维持 = 内部模型维持（以代谢为代价）
连线：认同是有热力学代价的信息结构
```

**适用层级**：主体层、认同层（含延伸至生命层的自组织系统）。  
**接口类型**：层级间结构同构，有完整数学形式化，神经科学有实验支撑。

**接口质量**：⭐⭐⭐⭐  
**失败条件**：若 Friston FEP 被证伪，或主体认同维持被证明不符合自由能最小化，映射失效；若 Markov 毯在神经系统中无法精确划定，接口降级为功能类比。  
**防漂移边界**：FEP 描述认同维持的信息-热力学机制，不替代主体的现象学描述，不声称意识 = 自由能最小化。

## 10. 希望层科学接口（方向已审批，待形式化）

**状态**：方向 ✅ APPROVED（2026-06-08，王俊华）；具体接口尚待形式化建立。  
**理论定义**：希望 = 主体在局限中仍开放于更高可能性的能力（非乐观情绪，非期待偏差，是结构性能力）。

**已排查的候选及其不足**：

| 候选 | 不足 |
|---|---|
| 认知探索（Friston epistemic foraging） | 目标是减少不确定性（收敛），与希望的"保持开放"方向相反 |
| 韧性理论（Holling panarchy） | 层级不匹配（生态系统层 vs 主体层），缺乏神经科学桥接 |
| 心理学乐观主义研究 | 描述期望偏差，不是结构性能力，与理论定义不符 |

**已审批的构造方向**：  
可能性空间熵（Possibility Space Entropy）——希望 = 主体对未来状态分布维持的香农熵高于当前信息所要求的最低值；即系统主动保留超出当前需要的可能性权重。

此定义若成立：
- 与 Landauer 的潜在连接：维持高可能性熵需要额外信息处理代价，希望是有热力学代价的结构性开放
- 需要形式化：定义"可能性状态分布"及其神经科学可测量对应
- 失败条件：若"可能性空间"无法被精确操作化，候选无效
