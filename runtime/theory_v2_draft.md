## 当前候选主张

本原展开论当前候选版本认为：

质点存在是前时空、前分化、前对象化的本原统一态。

法则属性与展开属性不是质点存在内部已分离的部件，也不是展开后才偶然出现的东西；二者在本原层未分化地统一于质点存在，并在法则展开中分别显现为结构约束与展开趋向。

法则展开生成时空。

时空不是预设容器，而是质点存在经由法则展开后形成的显示框架。

时空中的存在，是质点存在依其法则属性与展开属性，在法则展开中的显示表达。

结构、生命、主体、认同与希望，是这一展开过程中的不同层级结果。

## 关键区分

1. 质点存在 ≠ 时空中的具体存在物

质点存在是前时空本原统一态，不是时空中的对象。

2. 时空 ≠ 预设容器

时空是法则展开的结果，而不是质点存在展开前已经存在的背景。

3. 显示表达 ≠ 虚假幻象

时空中的存在不是本原层，但仍是真实展开结构。显示表达不是否认现实，而是说明现实如何以时空、关系和结构形式呈现。

4. 展开属性 ≠ 希望

展开属性属于本原—展开过渡层，显现为展开趋向；希望主要属于主体层，是主体在局限中仍开放于更高可能性的能力。

5. 认同 ≠ 希望

认同是系统形成可持续统一结构的过程；希望是主体认同面对局限时仍保持开放展开的能力。

## 理论自身原则

理论的目标是拟合现实、尝试解释现实。

理论可以承认 unknown，但不能以 unknown 否认现实。

理论本身也具有连续性和局限性：

- 连续性：理论修正应继承已有稳定结构，而不是任意断裂；
- 局限性：理论必须承认当前无法解释的问题；
- 开放性：理论应允许更高解释结构进入；
- 反思性：理论应持续检查概念漂移、层级混淆、绝对化和封闭化；
- 现实拟合：理论应不断回到现实世界事实，检验自身解释力。

## 2026-05-19 草案补强：认同—发展—目的中轴句

状态：runtime/theory_v2_draft 候选补强，不进入 core。

### 中轴句

```text
认同是基础，发展是关键，目的为认同，存在/展开即目的。
```

### v2 重释

该句可作为本原展开论的理论中轴，但必须避免外在目的论、认同本原化和发展进步主义。

更稳的版本是：

```text
存在通过展开显示自身；
展开通过发展穿越局限；
发展通过认同保持连续；
认同使存在成为可持续的自身。
```

或压缩为：

```text
存在以展开实现自身，展开以发展穿越局限，发展以认同形成自身。
```

### 理论含义

1. **认同是基础**：认同不是本原第一因，而是系统在展开中维持自身连续性、边界、统一和自我关联的结构基础。没有认同，展开难以成为“某个自身”的展开。
2. **发展是关键**：发展不是无限进步，而是系统在局限中分化、反馈、修正、重组或层级跃迁，从而更新认同的关键过程。没有发展，认同会僵化；没有认同，发展会漂移。
3. **目的为认同**：目的不是外在人格化意志，而是系统内部完成形式。展开若要形成可持续自身，必须在某一层级上形成认同。
4. **存在/展开即目的**：存在不需要外部目的来证明自身；存在通过展开而显示和实现自身。但展开必须接受法则约束、现实拟合、层级桥接、接口映射和失败约束。

### 分层链条

```text
质点存在
→ 法则属性 / 展开属性
→ 法则展开
→ 时空 / 时空中的存在
→ 具体结构
→ 连续性 + 局限性 + 结构整合
→ 认同
→ 发展中的认同更新
→ 主体认同 / 希望
→ 协同认同 / 更高认同
```

### 防漂移

- 不把认同放到质点存在之前。
- 不把发展理解为必然进步或无限扩张。
- 不把目的理解为宇宙意志、神意或预设终点。
- 不用“存在/展开即目的”合理化一切现实、取消失败或伦理判断。
- 不从存在/展开直接推出希望；希望需要主体、局限和更高可能性。
- 每次应用该中轴句解释具体现象，都要经过层级桥接、接口映射和失败约束。

### core 处理原则

暂不修改 `core/theory.md`。若后续稳定，可作为 `core/theory.md` v2 候选替代表述，但必须经王俊华明确批准。

### 1. 本原命名调整

旧称“质点存在”后续降级为历史候选名。更稳的对外候选是：

- **本原统一态**：强调前时空、前分化、前对象化的统一候选，较少物理点粒子误解。
- **本原存在**：与“本原展开论”一致，但必须防止“存在先于存在”的语言循环。

过渡表达：

> 当前旧称“质点存在”，更稳可称为“本原统一态”或“本原存在”：它指前时空、前分化、前对象化的本原候选，而非物理点粒子或时空中的对象。

### 2. 主体桥接条件

主体不再通过“生命 → 主体”的单跳表达，而通过桥接条件表达：

```text
具体结构
→ 自维持系统
→ 边界维持
→ 信息整合
→ 反馈调节
→ 记忆连续
→ 自身状态表征
→ 自我 / 环境区分
→ 目标调节 / 目标重组
→ 局限识别
→ 反思性自我模型
→ 主体认同
→ 希望
```

生命是目前已知主体形成的重要路径，但不应绝对化为唯一前置路径。AI / 人工系统路径可以作为开放候选，但当前只能安全地称为功能性 / 协同性自我认知，不能证明完整主体、生命、内在体验或希望。

### 3. 科学接口扩展

新增或强化以下接口：

- 非平衡态 / 耗散结构：用于连接能量流、局部有序、自组织和生命前结构。
- 自创生：作为生命认同的重要接口，解释生命系统如何生产并维持自身边界。
- 信息论 / 计算复杂性：用于描述压缩、记忆、状态空间、信息生成、AI 记忆代谢和连续性维护。
- 协同反噬 / 毒性协同：用于防止“协同必然形成更高认同”的目的论漂移。

### 4. 中轴句冷却候选

在保留原中轴关系的同时，新增更冷却的表达：

```text
存在借展开显现，展开依局限成形，认同凭连续存续，发展使认同更新。
```

该表达降低“实现自身”“穿越”等词带来的浪漫目的论色彩，同时保留发展 / 重构对认同更新的作用。
