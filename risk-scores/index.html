STS Score — 美国胸外科学会手术风险模型

适用于 CABG、主动脉瓣置换、二尖瓣手术及联合手术；预测30天死亡率及主要并发症（卒中、肾衰、再次手术等）

风险分层解读
STS PROM（预测30天死亡率）风险分层临床含义
< 4%低危常规外科手术准入，标准围术期管理
4%–8%中危需充分讨论利弊；可考虑 TAVR/TMVR 等介入替代
> 8%高危优先考虑介入治疗或保守策略；多学科心脏团队讨论

STS 同时提供8项并发症预测（卒中、肾衰、长时间通气、深部感染、再次手术等），可综合参考。

SYNTAX Score — 冠脉解剖复杂程度评分

基于冠状动脉造影的解剖学评分，指导三支病变 / 左主干病变 PCI vs CABG 血运重建策略选择

评分区间与血运重建策略推荐
SYNTAX 分值复杂程度三支病变推荐无保护左主干病变
≤ 22低复杂度 PCI 与 CABG 预后相似（可选 PCI） CABG / PCI 均可，参考 SYNTAX II
23–32中等复杂度 优先推荐 CABG；PCI 需 MDT 讨论 倾向 CABG；PCI 需 MDT 评估
≥ 33高复杂度 强烈推荐 CABG（I 类推荐） 推荐 CABG

依据：2018 ESC/EACTS 冠状动脉血运重建指南；SYNTAX 试验（Serruys et al. NEJM 2009）及长期随访数据。

基础设置
  • 冠脉优势类型（右优势 / 左优势 / 均衡型）
  • 需要处理的病变个数
每处病变逐一评分（12节段图）
  • 病变所在节段（按 SYNTAX 分段编号）
  • 完全闭塞（CTO）：闭塞时间、侧支等级
  • 三叉分叉（Medina 分型）
  • 开口病变（主动脉 - 冠状动脉开口）
  • 严重扭曲（病变前段 ≥ 2处弯曲 > 70°）
  • 长病变（> 20 mm）
  • 严重钙化
  • 血栓性病变
  • 弥漫病变 / 小血管（直径 < 2 mm）
SYNTAX Score 需逐段解剖评分，算法复杂，不支持本地准确计算。建议由经过培训的医师使用官方工具。

SYNTAX Score II

结合解剖学 SYNTAX 评分与8项临床因素，分别预测 PCI 和 CABG 的4年全因死亡率，支持个体化血运重建决策（Farooq V et al., Lancet 2013;381:639–650）

决策框架
比较结果推荐策略
CABG 预测4年死亡率 明显低于 PCI（差值 > 5%）推荐 CABG
两者预测死亡率 相近（差值 ≤ 5%）个体化决策（患者意愿、解剖可行性、合并症）
PCI 预测4年死亡率 明显低于 CABG（差值 > 5%）倾向 PCI（患者可能不耐受开胸手术）
SYNTAX Score II 主要在左主干病变和三支病变人群中验证，用于其他情况需谨慎。

NCDR CathPCI 风险评分

美国心脏病学会心导管数据库（ACC-NCDR）模型，预测 PCI 围术期院内死亡率及出血、急性肾损伤等主要并发症

院内死亡率风险分层
预测院内死亡率风险分层参考意义
< 1%低危常规 PCI，标准围术期管理
1%–3%中危充分告知患者风险；优化抗栓和支持治疗
≥ 3%高危充分知情同意；讨论 IABP / Impella 等辅助支持；评估 CABG 是否更安全
NCDR CathPCI 同时可预测出血并发症（BARC ≥ 3）风险，用于围术期出血 - 缺血平衡决策。
